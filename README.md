# Navigation and Sensor Fusion C++ Library
General navigation software library for sensor fusion. 

# Dependencies
* C++ 11 (or greater) <br />
* CMake (3.22.0 or greater) <br />
* Eigen (3.3 or greater) <br />

# Build
```
mkdir build
cd build
cmake ..
make
```
# Run Unit Tests
```
./test_navigation_library
```
